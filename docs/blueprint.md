# **App Name**: EduChain

## Core Features:

- User Role Management: Manages different user roles (Director, Counselor, Teacher, Student) with distinct permissions.
- School Cycle Creation: Allows the Director to create and manage school cycles and groups.
- Student Enrollment and Assignment: Enables Counselors to enroll students and assign them to specific groups.
- Timetable Management: Counselors can create and manage timetables, and assign subjects to teachers.
- Attendance Tracking: Teachers can record student attendance via mobile app.
- Grade Submission: Teachers can submit grades for students via mobile app.
- Student Dashboard: Students can view their timetable and grades through a mobile app.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and authority.
- Background color: Light blue (#E8EAF6), a desaturated version of the primary, to keep things calm and light.
- Accent color: Warm amber (#FFB300), analogous to blue, to highlight interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that's both modern and friendly.
- Use consistent and clear icons throughout the app for easy navigation.
- Web dashboard will follow a structured, data-rich layout, whereas mobile apps will feature simpler, focused layouts.
- Use subtle transitions and animations to enhance user experience and provide feedback on interactions.
